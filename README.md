# MySoftRender

# 1.简介
这是我对渲染管线的练习，内容基于对渲染流程的理解实现了基本的光栅化渲染器的一些常见特性，如下：  
-MVP矩阵变换  
-简单CVV剔除  
-视口渲染  
-光栅化  
-纹理采样  
后新加：  
-基本的光照模型  
-背面剔除  
-线框渲染  
-相机控制
